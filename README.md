# Finance Tracker

## Description
Finance Tracker is a comprehensive application that helps users manage their financial activities, including income, expenses, budget planning, and financial reporting. It provides an intuitive interface to facilitate users in tracking their finances effectively.

## Technology Stack
- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js
- **Database:** MongoDB
- **Frameworks:** Express.js

## Installation Instructions
1. Clone the repository: `git clone https://github.com/PrajwalShettyR/Finance_Tracker.git`
2. Navigate into the project directory: `cd Finance_Tracker`
3. Install dependencies: `npm install`
4. Set up your environment variables in a `.env` file.
5. Start the application: `npm start`

## Usage Examples
- Add a new income: Navigate to the "Income" section and add your income details.
- Track expenses: Use the "Expenses" section to log your spending.
- View reports: Access the "Reports" section to visualize your financial performance.
